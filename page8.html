SpeedRacer 2099

Overview

This project was created as part of a personal game jam in the summer of 2018. Me and my brother really love car racing games and I decided to build one so we could play it together. Some features are hover physics, lap counter, mini-map and explosions! Player 1 uses game-pad and Player 2 plays with the keyboard (Personal Choice).

Theme

The environment is set in a futuristic desert racing track.

Core Gameplay Mechanics

Some core mechanics are hover physics, lap counters, mini-maps and wrong-way detection.

Assets

This game is developed on Unity Engine 2018.1 with C#.
Art Assets DO NOT belong to me and are used for educational purposes only.

Target Platform

This game is developed for the PC platform with plans to add a Machine Learned AI car for solo-mode that the player can compete against.
