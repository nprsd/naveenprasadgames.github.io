Color Switch

Overview

This project is an individual 2 player game developed for Console Game Development I. It was designed to teach us advanced topics of Unreal Engine and UWP. We were also taught how to develop a game that meets TCR so we can publish to Xbox. The game is very simple, the player has to match the colors shown on screen with the corresponding color key. Player with the highest points wins!

Theme

Retro style button masher.

Core Gameplay Mechanics

This game requires game-pads so that the player can match the colors with the gamepad button color. The game is timed and at the end, the player with the highest points wins.

Assets

This game is developed on Unreal Engine UWP with C++ and Blueprints.

Target Platform

This game is developed for Xbox family of devices (Xbox One, Xbox One S and Xbox One X).
